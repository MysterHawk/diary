# Diary

This project is a digital diary that I'm building to organize my life.

It still in WIP but with the time I will add the functionality needed.

## Tech Stack

**Client:** React, Redux, TailwindCSS

**Server:** Node, Express


## Screenshots

## Features

- Light/dark mode toggle
- Live previews
- Fullscreen mode
- Cross platform
